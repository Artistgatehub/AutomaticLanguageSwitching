AutomaticLanguageSwitching Installation

The native host was installed automatically.

The unpacked Chrome extension folder was placed at:
%LOCALAPPDATA%\AutomaticLanguageSwitching\Extension

Finish setup in Chrome:

1. Open chrome://extensions
2. Enable Developer mode
3. Click "Load unpacked"
4. Select the folder above

The installer also opens that folder in Explorer so it is immediately visible and easy to choose.
The installer also tries to open chrome://extensions for you.
If that page does not open automatically, open it manually in Chrome.

Important:
- The native host registration is already installed for your Windows user
- The unpacked extension uses a fixed manifest key, so its extension ID stays stable across install locations
- Chrome still requires the final unpacked-extension step to be completed manually
